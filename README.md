# Chiedilo a Ivan

L'obiettivo di questo progetto si divide in due parti:

1. Il primo è necessario al fine del secondo in realtà, ma direi che può essere comunque un' evoluzione dell'Osservatorio. Si tratta di portare completamente in cloud il database di Upipa. Questa è una soluzione sicuramente più moderna, ma soprattutto mi permette di dare accesso ai dati a chiunque voglia e abbia l'autorizzazione per farlo. In particolare si può...
2. Sviluppare un semplice chatbot con accesso al nostro database in modo che possa rispondere a domande semplici sui dati che abbiamo a disposizione noi. Questo darebbe facile accesso ai miei colleghi a tutti i dati che l'osservatorio raccoglie, senza aver bisogno di chiedere a me idealmente, soprattutto per le cose più semplici. L'idea è quella poi di estendere un po' alla volta le funzionalità del chatbot, magari producendo qualche grafico e/o tabella.
